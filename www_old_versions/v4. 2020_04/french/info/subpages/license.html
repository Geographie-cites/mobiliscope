Codes, données et visuels sous licences libres

Le développement du Mobiliscope s’inscrit dans une démarche de science ouverte : nous faisons notre maximum pour que les données construites par l'équipe et affichées dans l'outil respectent les principes FAIR : « Faciles à trouver, Accessibles, Interopérables et Réutilisables ».

Code source

Le Mobiliscope est développé en HTML, CSS (framework mini.css), javascript et PHP. La géovisualisation a été codée en javascript et s’appuie, pour l'interactivité des cartes et graphiques, sur les libraires D3.js - développé par Mike Bostock - et Leaflet. La superposition des tuiles OSM et des cartes dessinées avec D3.js a été facilitée par l'utilisation du plugin Leaflet.D3SvgOverlay sous licence MIT.

Les données agrégées et pondérées (qui ne permettent pas la réidentification des personnnes enquêtées) sont stockées localement sous forme de fichiers geojson pour les données spatiales et csv pour les données attributaires sans géométrie. Ces données ont été automatiquement générées en amont grâce à deux scripts R : le premier transforme les données de déplacements issues des enquêtes en données de présences ; le second produit l’ensemble des fichiers nécessaires à la représentation (carto)graphique pour chaque indicateur.

Le code du Mobiliscope (comprenant les codes du logiciel et les programmes de traitements de données) est accessible sur un dépôt public github. Ce code est mis à disposition sous licence libre AGPL.

Cette licence AGPL autorise toute redistribution ou modification du code sous réserve qu’il demeure sous licence AGPL et que la source du code soit identifiée avec la mention : « Code sous licence AGPL issu du Mobiliscope (v4), un outil de géovisualisation de la ville à toute heure (https://mobiliscope.cnrs.fr) ».

Données

Les données de présence dans les villes françaises sont proposées au téléchargement (format .cvs et .geojson) sur le site du Mobiliscope, accompagnées d'un fichier résumant les termes de la licence ODbL et les sources à indiquer.

Pour télécharger les données, cliquez sur le bouton

  • soit au dessus de la carte centrale pour obtenir les valeurs agrégées par secteur selon l'indicateur et le mode de représentation sélectionné
  • soit à côté du graphique du bas pour obtenir les valeurs des indices de ségrégation pour l'ensemble de la région selon l'indicateur sélectionné et l'indice de ségrégation choisi.

Ces données sont mises à disposition sous licence libre ODbL. Elles sont donc librement réutilisables et modifiables sous réserve qu’elles demeurent sous licence ODbL et que les sources soient identifiées avec la mention : « Données sous licence ODbL issues du Mobiliscope (v4), un outil de géovisualisation de la ville à toute heure (https://mobiliscope.cnrs.fr), à partir des données initiales de [à compléter selon les sources indiquées sur la carte de la ville correspondante ou dans les termes de licence de réutilisation] ».

Visuels

Les visuels (cartes et graphiques) affichés dans le Mobiliscope sont mis à disposition sous licence libre creative commons CC-BY-SA.

Les visuels sont donc librement réutilisables sous réserve que les auteurs et les sources soient cités, par exemple avec la mention: « Visuels issus du Mobiliscope (v4), un outil libre de géovisualisation des villes à toute heure (https://mobiliscope.cnrs.fr) ».

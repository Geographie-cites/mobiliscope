Codes, visuels et données sous licences libres

Le développement du Mobiliscope s’inscrit dans les principes du FAIR : nous faisons notre maximum pour que les données construites par l'équipe et affichées dans l'outil soient « Faciles à trouver, Accessibles, Interopérables et Réutilisables ».

Code source

Le Mobiliscope est développé en HTML, CSS (framework mini.css), javascript et PHP. La géovisualisation a été codée en javascript et s’appuie, pour l'interactivité des cartes et graphiques, sur les libraires D3.js - developpé par Mike Bostock - et Leaflet. La superposition des tuiles OSM et des cartes dessinées avec D3.js a été facilitée par l'utilisation du plugin Leaflet.D3SvgOverlay sous licence MIT.

Les données (stockées localement sous forme de fichiers geojson pour les données spatiales et csv pour les données attributaires sans géométrie) sont générées automatiquement en amont grâce à deux scripts R : le premier transforme les données de déplacements issues des enquêtes en données de présences ; le second produit l’ensemble des fichiers nécessaires à la représentation (carto)graphique pour chaque indicateur.

Le code du Mobiliscope (comprenant les codes du logiciel et les programmes de traitements de données) est accessible sur un dépôt public github. Ce code est mis à disposition sous licence libre AGPL.

Cette licence AGPL autorise toute redistribution ou modification du code sous réserve qu’il demeure sous licence AGPL et que la source du code soit identifiée, par exemple avec la mention : « Code issu du Mobiliscope (v4), un outil libre de géovisualisation des villes à toute heure (https://mobiliscope.cnrs.fr) ».

Visuels

Les visuels (cartes et graphiques) affichés dans le Mobiliscope sont mis à disposition sous licence libre creative commons CC-BY-SA.

Les visuels sont donc librement réutilisables sous réserve que les auteurs et les sources soient cités, par exemple avec la mention: « Visuels issus du Mobiliscope (v4), un outil libre de géovisualisation des villes à toute heure (https://mobiliscope.cnrs.fr) ».

Données

Certaines données de présence agrégées par heure et par secteur peuvent être téléchargées (format .cvs et .geojson) sur le site du Mobiliscope (ainsi que la licence de réutilisation).

Ces données mises à disposition sous licence libre ODbL sont librement réutilisables et modifiables sous réserve qu’elles demeurent sous licence ODbL et que les sources soient identifiées, par exemple avec la mention : « Données sous licence ODbL issues du Mobiliscope (v4), un outil de géovisualisation de la ville à toute heure (https://mobiliscope.cnrs.fr), à partir des données initiales de [à compléter selon les sources indiquées sur la carte de la ville correspondante ou dans les tremes de licene de réutilsation] ».
